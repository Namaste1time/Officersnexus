Add Questions (Bilingual Format)

Format Instructions (Bilingual):

Enter each question in the following format for bilingual support:

Q. [English Question] / [Hindi Question]
A. [English Option 1] / [Hindi Option 1]
B. [English Option 2] / [Hindi Option 2]
C. [English Option 3] / [Hindi Option 3]
D. [English Option 4] / [Hindi Option 4]
ANS: A/B/C/D
EXP: [English Explanation] / [Hindi Explanation]

Separate questions with a blank line.

Example:

Q. What is the capital of France? / फ्रांस की राजधानी क्या है?
A. London / लंदन
B. Berlin / बर्लिन
C. Paris / पेरिस
D. Madrid / मैड्रिड
ANS: C
EXP: Paris is the capital of France / पेरिस फ्रांस की राजधानी है
